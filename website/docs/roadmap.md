# Roadmap

The future of Slides Plus.

## Version History

### v0.1.x - Foundation
- Initial release
- Core presentation features
- Basic themes and layouts
- Integrations (Mermaid, LaTeX, Excalidraw)

### v0.2.0 - Custom Themes (Planned)
- User-created themes
- Theme API for developers
- Custom CSS support

### v1.0.0 - Production Ready
- Complete test coverage
- Stable API
- Comprehensive documentation

## Feature Roadmap

### High Priority

| Feature | Status | Description |
|---------|--------|-------------|
| Custom themes | Planned | User-created CSS themes |
| Image backgrounds | Planned | Per-slide image backgrounds |
| Drag-to-reposition | Planned | Move elements in preview |
| Bug fixes | Ongoing | From user feedback |

### Medium Priority

| Feature | Status | Description |
|---------|--------|-------------|
| Element animations | Planned | Appear/fade individual items |
| Slide templates | Planned | Reusable slide patterns |
| Remote control | Planned | Control from phone |

### Lower Priority

| Feature | Status | Description |
|---------|--------|-------------|
| PPTX export | Considered | PowerPoint compatibility |
| Video embedding | Considered | Media support |
| Collaboration | Considered | Real-time editing |

## Release Process

1. Features freeze for release
2. Bug fix phase
3. Testing
4. Release candidate
5. Stable release

## Contributing to Roadmap

- Vote on features in [discussions](https://github.com/aqcohen/obsidian-slides-plus/discussions)
- Propose new features
- Help prioritize

## Long-term Vision

- Best-in-class presentation tool for Obsidian
- Community-driven development
- Sustainable maintenance
