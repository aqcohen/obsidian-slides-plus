- **Markdown-based** - Write presentations in plain Markdown
- **Live Preview** - See your changes in real-time
- **Custom Themes** - Full control over styling
- **Multiple Layouts** - Title, two-column, center, and more
- **Export to PDF** - Share your presentations offline
